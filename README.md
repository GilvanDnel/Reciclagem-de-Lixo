https://gilvandnel.github.io/Reciclagem-de-Lixo/

## Tecnologias

- HTML
- CSS
- JavaScript
- GitHub Pages

## Estrutura

- `Exercicios/public/index.html`: pagina principal
- `Exercicios/public/styles/main.css`: estilos
- `Exercicios/public/scripts/main.js`: interacoes
